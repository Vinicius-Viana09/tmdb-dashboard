## 📊 Sobre o Projeto

Este projeto é um **dashboard interativo de análise de filmes**, desenvolvido com Python e Streamlit, utilizando dados do TMDB (The Movie Database).

O objetivo é explorar padrões entre **receita, avaliação, popularidade e votos**, transformando dados brutos em insights visuais e interativos.

---

## 🎯 Objetivo

- Realizar análise exploratória de dados (EDA)
- Identificar relações entre sucesso comercial e avaliação
- Criar visualizações interativas para análise de filmes
- Praticar construção de dashboards com Streamlit

---

## 🛠️ Tecnologias Utilizadas

- Python
- Pandas (manipulação de dados)
- Streamlit (interface interativa)
- Plotly (visualização de dados)
- TMDB Dataset

---

## 📂 Dataset

O dataset utilizado contém informações de filmes do TMDB, incluindo:

- Título
- Orçamento
- Receita
- Avaliação média
- Número de votos
- Popularidade

Esses dados foram usados para análise comparativa e visualização de desempenho dos filmes.

---

## 📌 O que foi aplicado no projeto

- Limpeza e tratamento de dados com Pandas
- Criação de KPIs dinâmicos
- Construção de rankings interativos
- Visualização de dados com Plotly
- Filtros dinâmicos com Streamlit
- Storytelling com dados

---

## 🧠 Principais Insights

- Filmes com maior receita não são necessariamente os mais bem avaliados
- Popularidade não garante alta avaliação média
- Sucesso comercial e qualidade percebida não seguem relação direta
- Volume de votos não determina necessariamente a nota média
